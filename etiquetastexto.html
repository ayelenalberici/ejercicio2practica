Wikipedia es una enciclopedia libre,nota 2​ políglota y editada de manera colaborativa. Es un proyecto de crear una enciclopedia libre en la red. Cada uno puede aportar sus conocimientos sobre cualquier tema para crear una base de datos con toda la sabiduría humana.
